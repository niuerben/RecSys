提交用户推荐表

支持上传：

  • 单个最终 recommender.csv
  • 多轮 recommender_*.csv 的 ZIP

推荐表的表列格式：

user_id, recommendations1, recommendations2, ..., recommendations100

尚未选择文件

模型性能榜单

Model Recall@10 MRR@10 NDCG@10 Hit@10 Precision@10

评估协议

数据集：MovieLens-1M

数据划分：测评机从 rating 表中按用户随机化后 10% 作为测试集。提交推荐表无需考虑数据划分。

正样本定义：rating >= 3；rating < 3 的 item 仍是合法候选，但不算命中。不在测试集但在 rating 表中的 item 被 mask，不在 rating 表中的 item 被视为负样本

评测规则：提交 Top100 推荐，mask 后取过滤后的 Top10 计算指标。

用户数为 6040。支持空用户。每个用户至多推荐 top100 个物品。
